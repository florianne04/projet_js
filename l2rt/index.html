DEVOIR JAVASCRIPT - DOM & ÉVÉNEMENTS

Thème : Coupe d'Afrique des Nations (CAN)

Objectif

Rendre la page web interactive en utilisant JavaScript, le DOM et les événements. Le HTML et le CSS sont déjà fournis.

Travail demandé

1. Équipes participantes

  • Un tableau JavaScript contenant les pays qualifiés à la CAN est fourni.
  • Charger dynamiquement ce tableau dans la section « Équipes participantes ».
  • Aucune équipe ne doit être écrite directement dans le HTML.

2. Ajouter un match

Dans le formulaire Ajouter un match :

  • Équipe 1 et Équipe 2 ne doivent jamais être identiques.
  • Équipe 1 et Équipe 2 doivent obligatoirement figurer dans la liste des équipes participantes.
  • Si une condition n'est pas respectée, le match n'est pas ajouté.
  • Si tout est correct, le match est ajouté dans le tableau Matchs.
